## Описание задания
Изучить концепции ленивых вычислений, протоколы итерации и механизмы асинхронного/двустороннего взаимодействия в Python на примере числовой последовательности Фибоначчи.

### Задачи работы:
1. Реализовать генерацию списка чисел Фибоначчи с помощью **итераторов** двумя способами (класс-итератор и генераторная функция).
2. Реализовать класс-итератор `FibonacchiLst` для фильтрации элементов из произвольного списка, оставляя только числа, принадлежащие ряду Фибоначчи.
3. На основе предоставленного шаблона кода разработать **сопрограмму (корутину)** `my_genn`, позволяющую динамически по переданному значению $n$ через метод `.send(n)` генерировать и возвращать список элементов числового ряда Фибоначчи.
